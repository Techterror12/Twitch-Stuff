# Install

Just

go get github.com/ThyCorp/Twitch-Stuff

Then
cd  home/[Your_Home_Dir_Name]/go/src/github.com/ThyCorp/Twitch-Stuff/Thy-Twitch-Bot/src

Then
go run main.go run

The run At The End Is Required It Is The Option To Run The Bot
